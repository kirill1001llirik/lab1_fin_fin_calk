Запуск готовой версии на Windows:

1. Откройте файл run_financial_calculator.bat двойным щелчком.
2. В браузере откроется финансовый калькулятор.
3. Чтобы остановить приложение, закройте окно консоли или нажмите Ctrl+C.

Если браузер не открылся автоматически, скопируйте адрес из консоли. Обычно это:
http://127.0.0.1:8000/
