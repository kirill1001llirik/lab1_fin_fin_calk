# Финансовый калькулятор

Лабораторная работа: финансовый калькулятор для вычисления выражения из четырех чисел с фиксированной запятой.

Студент: Наркевич Кирилл Павлович, 3 курс, 2 группа, 2026 год, Минск.

## Возможности

- ввод четырех положительных или отрицательных чисел;
- выражение вида `Число1 op1 (Число2 op2 Число3) op3 Число4`;
- выбор операций: сложение, вычитание, умножение, деление;
- приоритет скобок для второго и третьего операнда, затем обычный математический приоритет операций;
- промежуточное округление до 10 знаков после точки по математическим правилам;
- сообщение об ошибке при делении на 0;
- поддержка точки и запятой как разделителя дробной части;
- поддержка корректных пробелов между тысячами: `1 234 567.89`;
- расчет в диапазоне от `-1000000000000.0000000000` до `1000000000000.0000000000`;
- сообщение о переполнении входных, промежуточных и итоговых значений;
- запрет экспоненциальной записи вроде `123e+2`;
- вывод результата с пробелами по тысячам, точкой как разделителем и без незначащих нулей;
- отдельный вывод результата, округленного до целого: математически, бухгалтерски или усечением;
- поддержка `Ctrl+C`, `Ctrl+V`, `Ctrl+X`, `Ctrl+A` в полях ввода средствами браузера.

## Запуск

Нужен только Python 3. Приложение запускает локальный веб-сервер и открывается в браузере.

```bash
python run.py
```

На Windows можно также запустить файл:

```bash
release\run_financial_calculator.bat
```

Если браузер не открылся автоматически, перейдите по адресу, который будет показан в консоли, обычно это `http://127.0.0.1:8000/`.

Чтобы остановить приложение, нажмите `Ctrl+C` в окне консоли.

## Проверка

```bash
python -m unittest discover -s tests
```

## Создание exe на Windows

Если нужен отдельный исполняемый файл, установите PyInstaller и выполните:

```bash
python -m pip install pyinstaller
python -m PyInstaller --onefile --windowed --name financial_calculator run.py
```

Готовый файл появится в папке `dist`.
